# RESTful API Design & Swagger

A Spring Boot REST API for managing customer data with Oracle XE database and Swagger UI documentation.

## Technologies
- Java 21, Spring Boot 3.3.0, Spring Data JPA, Oracle XE, Lombok, Springdoc OpenAPI

## Endpoints
- `POST /api/customers` – Create customer
- `GET /api/customers` – Get all customers
- `GET /api/customers/{id}` – Get by ID
- `PUT /api/customers/{id}` – Update customer
- `DELETE /api/customers/{id}` – Delete customer

## Swagger UI
http://localhost:8080/swagger-ui/index.html
